LUMOS: A LIGHT FOR ALL VOICES

LUMOS IS A WEB APPLICATION THAT REINVENTS SEARCH BY ILLUMINATING UNIQUE PERSPECTIVES. LUMOS CLUSTERS SEARCH RESULTS INTO SIMILAR VIEWPOINTS, ALLOWING USERS INTUITIVELY EXPLORE THE SEARCH SPACE. ADDITIONALLY, LUMOS CALCULATES A CREDIBILITY SCORE FOR EACH WEBPAGE TO COMBAT MISINFORMATION. LUMOS IS BUILT WITH PYTHON, REACT, AND JAVASCRIPT.

SEC Alumni Hub

Alumni Hub is a member-focused platform that acts as a supportive network between current and former SEC members. Alumnihub builds a constructive ecosystem where members can seamlessly seek advice, foster mentorship relationships, and build a network that transcends academic boundaries! Alumni Hub is built with React, Express, MongoDB, and Firebase.

Parasite Paradise

Parasite Paradise is a 2D arcade survival game where you play as a parasite that must survive in a hostile environment. Through this project, I Implemented AI state machines, enabling enemies to dynamically switch between behaviors, optimized projectile mechanics with object pooling, and managed a team of developers, story writers, digital artists, and sound engineers. Parasite paradise is built with C# and Unity.
